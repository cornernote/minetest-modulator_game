####################
###ABOUT THIS MOD###
####################
Tired of building boring streets of cobble or stone? This mod adds asphalt to minetest, which you can use to create asphalt blocks, asphalt blocks with side lines, with middle lines and lines for edges. It depends on no other mod so don't need to worry about errors when updating other mods. It is very simple, but I hope you'll have fun! :)
