Description
----------------------------------


Chests with slots that never run out!  

Name your chest slots so you can easily find your items.

Last accessed slot appears at the top of the list.



----------------------------------
Craft Guide
----------------------------------


CCC  <- Infinite Chest
C C      C = default:chest
CCC


CCC  <- Infinite Locked Chest
CMC      C = default:chest_locked
CCC      M = default:mese
